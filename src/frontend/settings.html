Access Locks (Environment)

Page protection is controlled in your .env file and applied at container start.

HAMVOX_PROTECT_PUBLIC: lock all public pages when true.
HAMVOX_PROTECT_TRANSCRIPTS: lock only the Transcripts page/API when true.
HAMVOX_PROTECT_LOGS: lock only the Logs page/API when true.
